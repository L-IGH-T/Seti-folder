# seti-icons

This is an icon theme with folder that uses the icons from [`seti-ui`](https://github.com/jesseweed/seti-ui).

## Usage 

    1. Open the command list (Press F1 or Ctrl + Shift + P)
    2. Select `File Icon Theme`
    3. Select `Seti folder`
